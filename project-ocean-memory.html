Ocean Memory Timescale Analysis

CESM1 · daily SST · 1980–2020 · Python/xarray · autoregressive modeling

In this project, I analyzed CESM1 high-resolution daily sea surface temperature (SST) data from 1980–2020 to explore the “memory” of the upper ocean. Using Python/xarray, I first detrended the SST time series to isolate internal variability, then applied autoregressive modeling (AR(1)) to estimate thermal memory timescales.

These memory estimates help us understand how long surface temperature anomalies persist, which has implications for predictability, marine heatwaves, and how different regions respond to climate forcing. I’m especially interested in how memory varies across basins and along western boundary currents, where dynamical processes are intense.

This project strengthened my comfort with working on large climate datasets, translating “ocean memory” from a conceptual idea into something you can actually compute and visualize pixel by pixel.

Details

  • Dataset: CESM1 high-resolution daily SST (1980–2020)
  • Tools: Python, xarray, NumPy, Matplotlib
  • Methods: detrending, AR(1) fitting, spatial mapping of timescales
